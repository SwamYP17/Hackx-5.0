These are the following requirements:

numpy==1.23.5
pandas==2.0.3
tensorflow==2.13.0
keras==2.13.1
opencv-python==4.8.0
scikit-learn==1.3.0
matplotlib==3.7.2
joblib==1.3.2

Explanation of the Packages:

numpy: For numerical operations, such as matrix manipulations.
pandas: To work with the dataset (CSV) and manage data frames.
tensorflow: The main deep learning framework to build and train the CNN model.
keras: High-level API for building and training the neural networks.
opencv-python: For image processing and reading images.
scikit-learn: To assist in metrics and model evaluations, if needed.
matplotlib: For visualizing images and plots during model training.
joblib: For saving and loading models efficiently (used in saving image_model.pkl).
